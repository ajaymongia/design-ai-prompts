# Design AI Toolkit

> A curated collection of AI prompts, workflows, templates, and best practices for UX/UI designers using Figma AI, Claude, and other AI-powered design tools.

---

# 🚀 Overview

**Design AI Toolkit** is a centralized repository of production-ready AI prompts designed to accelerate every stage of the product design process.

Whether you're conducting UX research, creating wireframes, building design systems, or preparing developer handoff documentation, this repository provides reusable prompts that help you produce consistent, high-quality outputs with AI.

The prompts are optimized for:

* 🎨 Figma AI Agent
* 🤖 Claude
* 💬 ChatGPT
* 📝 Any LLM that supports prompt-based workflows

# 📝 Prompt Structure

Every prompt follows the same format.

```markdown
# Title

## Purpose

## When to Use

## Inputs

## Prompt

## Expected Output

## Tips

## Example
```

---

# 📌 Example

```text
Title
Create User Personas

Purpose
Generate realistic personas for a new product.

When to Use
Discovery Phase

Prompt
Act as a Senior UX Researcher...

Expected Output

• Persona Cards
• Goals
• Pain Points
• Motivations
```

# 📖 Best Practices

* Write clear objectives.
* Provide enough context.
* Replace placeholders before running prompts.
* Review AI output before implementation.
* Validate recommendations with users and stakeholders.
* Keep prompts modular and reusable.
* Maintain consistency across projects.
