# Design AI Prompts

A curated collection of production-ready AI prompts for UX/UI Designers.

## Supported AI
- Claude
- ChatGPT
- Figma AI Agent

## Repository Structure

```
design-ai-prompts/
├── README.md
├── PROJECT_BRIEF.md
├── PROMPT_GUIDELINES.md
└── prompts/
    ├── design-system/
    ├── user-flow/
    ├── information-architecture/
    ├── competitor-analysis/
    ├── wireframe/
    ├── visual-design/
    └── moodboard/
```

Fill **PROJECT_BRIEF.md** once, then use the prompts inside each folder.
